# sound_volume_analyzer_lcd_i2c

Скетч для Arduino Uno:

- читает аналоговый звуковой датчик на пине A0;
- считает абсолютный уровень громкости по значению `analogRead(A0)`;
- калибрует базовый уровень шума при запуске;
- сглаживает показания скользящим средним за 10 измерений;
- выводит процент громкости и шкалу на LCD1602 через I2C;
- продолжает работать без дисплея и пишет значения в Serial Monitor.

## Файлы

- `sound_volume_analyzer_lcd_i2c.ino` - настройки, `setup()`, `loop()` и общий сценарий работы.
- `Sound.ino` - чтение аналогового сигнала и расчет уровня громкости.
- `Display.ino` - инициализация LCD и вывод числового уровня со шкалой.
- `Checks.ino` - проверка I2C/LCD и индикация активности через LED.

## Подключение

Пины и адрес дисплея вынесены в константы в начале файла `sound_volume_analyzer_lcd_i2c.ino`:

- `SOUND_SENSOR_PIN`
- `STATUS_LED_PIN`
- `LCD_I2C_ADDRESS`

Звуковой датчик:

- VCC -> 5V
- GND -> GND
- AO -> A0 (`SOUND_SENSOR_PIN`)

LCD 1602 I2C:

- VCC -> 5V
- GND -> GND
- SDA -> A4
- SCL -> A5

## Как считается громкость

Алгоритм:

- при запуске в течение `CALIBRATION_DURATION = 1000` мс измеряется базовый уровень датчика;
- во время калибровки рядом с датчиком желательно соблюдать тишину;
- каждые `SENSOR_READ_INTERVAL = 100` мс читается абсолютное значение `analogRead(A0)`;
- одно значение находится в диапазоне `0..1023`;
- последние `AVERAGE_SAMPLE_COUNT = 10` значений хранятся в кольцевом буфере;
- скользящее среднее считается за `10 * 100 мс = 1 секунда`;
- из среднего значения вычитаются базовый уровень и `NOISE_REDUCTION = 5`;
- очищенный уровень переводится в проценты от `0` до `100`.

Экран обновляется после каждого нового измерения, то есть раз в 100 мс.

## Дисплей

Первая строка:

```text
Vol: 42% L:430
```

`L` - это очищенный уровень: среднее значение `analogRead(A0)` за последнюю секунду минус базовый уровень, найденный при калибровке, и минус `NOISE_REDUCTION`.

Если до калибровки в тишине было `L:131`, после калибровки этот уровень должен стать близким к `0`. Если во время запуска был шум, программа примет этот шум за новый базовый уровень; в таком случае перезапустите Arduino в тишине.

Вторая строка:

```text
████████
```

Шкала занимает всю ширину LCD1602. Для более плавного заполнения используются пользовательские символы LCD.

## Библиотеки Arduino IDE

Установите через Library Manager:

- LiquidCrystal I2C

Если дисплей не показывает текст, проверьте I2C-адрес. В скетче указан частый адрес `0x27`; иногда используется `0x3F`.
