# wemos_oled_weather

Скетч для Wemos D1 mini / ESP8266 со встроенным I2C OLED 128x64.

Что делает:

- подключается к WiFi;
- получает погоду OpenWeatherMap раз в 5 минут;
- показывает температуру, влажность, давление, ветер, направление ветра и описание погоды;
- считает среднюю температуру и давление за 15 минут;
- переключает страницы OLED каждые 4 секунды;
- сохраняет последние успешные данные на дисплее, если очередной API-запрос завершился ошибкой.

## Подключение дисплея

По умолчанию используется SSD1306 128x64 по I2C:

```cpp
const byte OLED_I2C_ADDRESS = 0x3C;
const byte I2C_SDA_PIN = 4; // GPIO4 = D2
const byte I2C_SCL_PIN = 5; // GPIO5 = D1
```

Если встроенный дисплей на плате использует другой адрес, попробуйте `0x3D`.

## Настройки

В начале `wemos_oled_weather.ino` задайте:

- `WIFI_SSID`
- `WIFI_PASSWORD`
- `OPENWEATHER_API_KEY`
- `WEATHER_LATITUDE`
- `WEATHER_LONGITUDE`
- `WEATHER_PLACE`

Погода запрашивается на английском (`lang=en`), потому что стандартный шрифт `Adafruit_SSD1306` нормально поддерживает ASCII, но не кириллицу.

## Интервалы

```cpp
const unsigned long WEATHER_UPDATE_INTERVAL = 5UL * 60UL * 1000UL;
const unsigned long AVERAGE_WINDOW = 15UL * 60UL * 1000UL;
```

При обновлении погоды раз в 5 минут среднее за 15 минут считается по 3 последним успешным выборкам.

## Страницы OLED

Страница 1:

- текущая температура;
- влажность;
- давление.

Страница 2:

- средняя температура за 15 минут;
- среднее давление за 15 минут;
- количество накопленных выборок.

Страница 3:

- скорость ветра;
- направление ветра в градусах и буквенно (`N`, `NE`, `E`, ...);
- описание погоды.

Нижняя строка всех страниц показывает статус последнего обновления API: `OK` или ошибку (`ERR HTTP 401`, `ERR JSON error`, `ERR WiFi offline` и т.п.).

## Обработка ошибок

Если WiFi отключен, скетч пытается переподключаться раз в 10 секунд.

Если OpenWeatherMap вернул ошибку или JSON не разобрался:

- старые успешные данные остаются на экране;
- строка статуса показывает последнюю ошибку;
- новая выборка не попадает в среднее.

## Библиотеки Arduino IDE

Установите поддержку ESP8266 Boards.

Используемые библиотеки:

- ESP8266WiFi
- ESP8266HTTPClient
- Wire
- Adafruit GFX Library
- Adafruit SSD1306
- ArduinoJson

`Adafruit GFX Library`, `Adafruit SSD1306` и `ArduinoJson` установите через Library Manager.
